# 🏠 House Price Prediction

## 📌 Overview

This project predicts house prices using Machine Learning based on various features like LotArea, OverallQual, and YearBuilt etc.

---

## 🚀 Features

* Predict house prices accurately
* Data preprocessing and cleaning
* Model training using regression algorithm
* Easy-to-use prediction system

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

---

## 📂 Project Structure

* `house_pricing_predictor.ipynb` → Model training & analysis
* `housing_price_model.pkl` → Trained model
* `submission.csv` → Output predictions

---

## ⚙️ How to Run

1. Clone the repository
2. Install dependencies
3. Run the notebook

---

## 📊 Output

The model predicts house prices based on input features.
